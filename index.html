Melbourne Iguana Avoidance Map

Description of Map: In light of a recent space iguana attack on Melbourne Australia, this map has been created to allow those within the city to avoid the attackers

Description of how the map works: Displaying both the city of Melbourne and the iguana attack land zones, a user may click on the map to determine if they are in danger of attack.

  • See The Map: Iguana Avoidance Map
  • See the code for this map hosted on Github

United States Power Plants

Description of Map: Display of United States power plants scaled and colored according to their output and type.

Description of how the map works: A user may select the types of power production displayed and click on the map to show all the plants producing this type within 500km of this point. If a user double-clicks, the radius is reduced to 200km.

  • See The Map: Power Plant Map
  • See the code for this map hosted on Github

Kentucky Unemployment by County: 2000 – 2013

Description of Map: Drawing on thirteen years of county unemployment data, this map allows a user to see how this variable has changed over time.

Description of how the map works: With multiple layers of interactivity, the user can select specific counties to display data for, as well as see the overall trend in relation to the rest of the state.

  • See The Map: Unemployment Map
  • See the code for this map hosted on Github

Percent of Arable Land Cover in Africa: 2005 – 2012

Description: This map display the percent of arable land comprising a country's land cover.

Description of how the map works: The user is enabled to see how arable land has changed over the period between 2005 - 2012. This is espeially helpful in determining large scale trends and choosing countries with the most fertile land to provide insurance assets to.
